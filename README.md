# NavigationView

#### SwiftUI Navigation 공부
##### `NavigationView`
- `NavigationLink`를 이용하여 화면 이동
- `EnvironmentObject` 은 데이터전달에 용이
- `ObservableObject`를 `conform`하는 객체는 선언할때 `class`로  선언
- `SwiftUI` 프레임워크 안에 `Combine`이 `import` 되어있음
